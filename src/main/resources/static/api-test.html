Capstone API Test

로그인 → 세션 시작 → mp3 업로드 → AI 분석 → 결과 조회 순서로 바로 실험할 수 있게 만든 테스트 페이지입니다.

빠른 실험 순서

  1. 이메일과 비밀번호를 입력하고 로그인하거나 회원가입합니다.
  2. 레벨을 고르고 학습 세션 시작을 누릅니다. 첫 문제의 questionId와 expectedText가 자동으로 채워집니다.
  3. mp3 파일을 업로드합니다. 성공하면 uploadId가 자동 저장됩니다.
  4. AI 분석 요청을 누르면 Spring이 FastAPI로 전달하고 결과를 저장합니다.
  5. 마지막에 결과 상세 조회, 세션 피드백 조회로 저장 결과를 확인합니다.
Swagger UI: /swagger-ui/index.html

Session ID 없음
Upload ID 없음
현재 문제 없음 

    


    

      

        
1. 로그인 / 회원가입

        
처음이면 회원가입, 계정이 있으면 로그인만 하면 됩니다.

        /api/auth/signup, /api/auth/login

        
        

        
        

        
        

        

          
          
          
        

        

      


      

        
2. 학습 세션 시작

        
세션을 시작하면 문제 5개가 내려오고, 첫 문제의 questionId와 정답 문장이 자동으로 분석 영역에 채워집니다.

        /api/learning/sessions

        
        

        

          
          
          
        

        

      


      

        
3. mp3 업로드

        
mp3 파일만 업로드할 수 있습니다. 성공하면 uploadId가 자동 저장됩니다.

        /api/media/audio

        
        

        

          
          
        

        

      


      

        
4. AI 분석 요청

        
프론트가 실제로 쓰는 핵심 API입니다. sessionId, questionId, expectedText, frames를 보내면 Spring이 FastAPI로 전달합니다.

        /api/media/audio/{uploadId}/analyze

        
        

        
        

        
        

        
        

        
        

        

          
          
        

        

      


      

        
5. 결과 확인

        
분석이 끝나면 결과 상세와 피드백을 바로 확인할 수 있습니다. 세션 목록, 대시보드도 함께 볼 수 있습니다.

        

          
          
          
          
          
        

        

      


      

        
선택: 커리큘럼 조회

        
문제 목록을 레벨 기준으로 따로 보고 싶을 때만 쓰면 됩니다.

        /api/curriculum/levels/{level}/lessons, /api/curriculum/lessons/{lessonId}
